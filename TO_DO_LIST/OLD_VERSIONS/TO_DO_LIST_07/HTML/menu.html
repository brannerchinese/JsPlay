Functionality
  1. Button that fades in and out on hover. Different speeds for each.
  2. Content entered in form field appears at head of To-do list.
  3. Submit either with button or carriage return.
  4. Focus always in form field if form field visible.
  5. To-do list items are sortable.
  6. Double-clicking on item moves it from To-do list to Done list or back. Moved item explicitly removed from original list.
  7. Use of Bootstrap "modal": content overlay.
  8. Modal content loaded from external file.
    Tools used
  1. val()
  2. prepend()
  3. remove()
  4. dblclick()
  5. fadeTo()
  6. hover()
  7. focus()
  8. named function: placed before where it is called
  9. parents().eq(n): Nth parent of element
  10. attr()
  11. load()
  12. sortable()
  13. literal object used for toggling of IDs
  14. element specified by parent (e.g., $('#list ol'))
